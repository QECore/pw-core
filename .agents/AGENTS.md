# PW-Core Project Management & Issue Rules

- Bugs receive the `bug` label.
- Features receive the `feature` label.
- Enhancements receive the `enhancement` label.
- Documentation issues receive the `docs` label.
- New issues are automatically added to the `PW-Core Roadmap` project.
- Status defaults to `Backlog`.
- Priority defaults to `Medium` if not specified.

## Allowed Commands for Copilot Agent

When commanded by the user, the agent can structure and execute project management tasks following these formats:
- **"Create a feature for table assertions."**
- **"Create a bug for IntelliSense ordering."**
- **"Create an enhancement for locator filters."**
- **"Create documentation for MCP integration."**
